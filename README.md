## ✨ Features

This landing page isn't just a website; it's a visual journey. It features advanced CSS rendering, precise positioning, and scroll-triggered animations that bring the Stark legacy to life.

- 🌌 **Cinematic Scroll Sequences:** Smooth parallax effects and scroll-locked dramatic reveals powered by GSAP.
- 📺 **J.A.R.V.I.S. UI Aesthetics:** High-tech holographic grids, rotating arc reactor loaders, and glowing neon accents.
- ⚡ **Dynamic Glitch Typography:** Custom CSS keyframe animations creating unstable, futuristic text effects on load.
- 🤖 **Interactive 'Hall of Armor' Accordion:** A sleek, expandable horizontal accordion showcasing the evolution of the Iron Man armors.
- 💔 **The End-Game Finale:** A visceral two-part scrolling conclusion:
  - **The Arc Reactor Legacy:** Floating, interactive memory nodes containing emotional quotes surrounding the original "Proof That Tony Stark Has A Heart" reactor.
  - **The Snap:** A dramatic, scroll-pinned zoom-out sequence revealing the decisive Infinity Stone snap, complete with lighting flashes and colossal typography.

---

## 🛠️ Built With

*   **HTML5 & CSS3:** For robust structure and advanced futuristic styling (CSS Grid, Flexbox, custom glowing filters, blending modes).
*   **Vanilla JavaScript:** Lightweight DOM manipulation and event handling for interactive elements like the tech tooltips and accordion logic.
*   **[GSAP 3](https://greensock.com/gsap/):** The industry standard for robust, high-performance web animations. 
*   **[ScrollTrigger](https://greensock.com/ScrollTrigger/):** Used extensively to pin sections, scrub timelines over scroll distances, and trigger complex entrance sequences.

## 🚀 Getting Started

To run this cinematic experience locally on your machine:

1.  **Clone the repository**
    ```sh
    git clone https://github.com/your_username/IronMan-Experience.git
    ```
2.  **Open the project**
    Simply navigate to the cloned folder and open `index.html` in any modern web browser. No complex build tools or servers are required!
